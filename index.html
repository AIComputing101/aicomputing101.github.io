How to Get Started

1

Fork & Clone

Start by forking the repository you want to contribute to, then clone it to your local machine.

2

Create a Branch

Create a new branch for your feature or fix. Use descriptive names like `feature/add-quantum-algorithm`.

3

Make Changes

Write clean, well-documented code. Follow our style guide and include tests where applicable.

4

Submit PR

Push your changes and open a pull request with a clear description of what you've done and why.
